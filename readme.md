Projektbeschreibung:
Die Idee des Projektes lässt sich auf die 1996er Version des Spieles Pokemon zurückführen. Natürlich hebt sich ‚Monsters‘ deutlich von diesem ab.
Man kann Monster fangen und mit ihnen kämpfen. Ab einer bestimmten Anzahl an gewonnenen Kämpfen kann das Monster aufleveln und wird stärker. Das Ziel dieses Spieles ist alle im Spiel vorkommenden Monster zu fangen und sich mit ihnen von Level zu Level zu kämpfen. Derzeit wurden nur 3 Levels programmiert.

Weitere Mitwirkung:
Die Idee zur Lösung der Bewegung des Charakters und dem Culling (man sieht immer nur einen Teil der ganzen Map), wurde auf den ersten 3 Tutorials „Javascript & Canvas 2d“-Reihe auf dem YouTube Channel „Technologies4.me“ erlernt.
Alle eingebundenen Bilder wurden von meiner Partnerin Iris Schlager entworfen und umgesetzt.
Die durchgehende Background-Musik wurde von einer Freundin (Marie Heng) zur Verfügung gestellt.
Alle weiteren musikalischen Stücke wurden von verschiedenen Websites welche copyright freie Musik anbieten verwendet.
